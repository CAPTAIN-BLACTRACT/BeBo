# BeBo

A **simple beat-making sequencer** built with **Angular** — designed to let you create beats visually and interactively.

---

##  Features

- Grid-based step sequencer UI
- Play/stop controls
- BPM adjustment
- Multiple tracks
- Interactive beat making

---

##  Tech Stack

This project is built with:

- **Angular** (frontend framework)
- **TypeScript**
- **HTML/CSS**
- Browser **Web Audio API** (for sound)

---

## Getting Started

Clone the repo:
```bash
git clone https://github.com/CAPTAIN-BLACTRACT/BeBo.git
```

Install dependencies:

```bash
cd BeBo
npm install
```

Run the development server:
```bash
ng serve
```

##  Project Structure

src/app/ – main application code

src/assets/ – static assets (styles, audio files, etc.)

src/index.html – entry HTML
